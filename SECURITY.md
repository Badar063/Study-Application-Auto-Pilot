# Security & Privacy

## Data Protection
- No personal data is stored in the repository
- All user configurations are local-only
- RSS feeds are from public academic sources only
- Identifiers are hashed for privacy

## Safe Usage
1. Never commit personal configuration files
2. Use the provided config template
3. Review generated content before sending
4. Be respectful to university servers

## Privacy Features
- URL hashing for source identification
- No personal information in logs
- Automatic cleanup of temporary files
- Configurable privacy settings
